Blogs Posted

The Blog Posts section summarizes articles that share knowledge and lessons learned during the Fraud Detection on AWS project. The blog posts are aligned with the personal project, covering system architecture, cost optimization, and the design of a real-time pipeline on AWS.

Blog list

Blog 1 - Credit Card Fraud Detection with Machine Learning on AWS

  • Main content: Designing a real-time credit card fraud detection system on AWS, including the Training Zone, Real-time Zone, deployment lessons, cost optimization, and future development directions.

Blog 2 - How Does Amazon Automate Workforce Planning with Serverless?

  • Main content: Amazon’s ALAP case study, using Serverless and Event-Driven architecture with S3, Glue, Aurora PostgreSQL, EventBridge, SQS, Lambda, DynamoDB, Cognito, and CloudFront to automate workforce planning.

Blog 3 - Running Serverless Apache Spark with Amazon Athena

  • Main content: Sharing how Amazon Athena supports serverless Apache Spark, enabling Spark Sessions through Spark Connect without managing EMR or Spark Clusters, while taking advantage of S3, Glue Data Catalog, Lake Formation, and Airflow.
